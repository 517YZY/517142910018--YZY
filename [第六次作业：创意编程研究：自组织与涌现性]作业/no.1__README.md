第六次：
本代码通过运用粒子系统创作模拟自然风向的作品，其中包含三个粒子系统（三种颜色）。基于这个代码，很简单的就可以根据自己想要的颜色种类和数量进行参数更改而实现。
Particle类具有速度、加速度、生命周期life等概念，
ParticleSystem类用于管理粒子的，可以添加多个粒子系统。其中包含着继承思想，可以通过一个系统管理不同的粒子。
通过覆盖调整图层透明度得到消隐拖尾的效果。
