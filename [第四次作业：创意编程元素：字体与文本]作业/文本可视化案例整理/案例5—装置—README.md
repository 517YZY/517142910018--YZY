案例5：
项目名称：Visual character installation
艺术家：Joseph Egan
来源：https://www.topys.cn/article/6415
概要描述：来自伦敦的图形设计师 Joseph Egan 与Chelsea 艺术与设计学院的同事们共同创作了这系列惊人的3D 错视装置艺术，运用〝Anamorphic〞的变形文字技巧，他们将学校的基础学科教室以及办公室整个变成装置艺术的展示场。
